# 火柴人战争

[中文](README.md) | [English](README.en.md)

《火柴人战争》是一款面向观众的“赛博斗蛐蛐”自动战争 Demo。玩家只需在开始界面点击一次抽签，四个阵营会随机选出两方，随后双方通过一体式双层弹珠机自动积攒兵力、释放技能并攻击敌方要塞。

[在线体验](https://stickman-war-demo.pages.dev/)

## 当前玩法

- 四个阵营：赤焰军、霜蓝卫、翠核同盟、鎏金王庭；单场模式随机抽取两个阵营并决定左右站位，锦标赛模式则随机生成两场半决赛签位。
- 左右两台机器每秒自动发射 1 颗弹珠。上层按照实际物理落点进入兵种、分裂或强化池，当前池宽比例为 `75% / 10% / 15%`。
- 单场战斗时间达到 `05:00` 后，双方上层主炮进入火力超频，基础产珠速率同步提高到每秒 2 颗；下一场恢复为每秒 1 颗。
- 分裂会把原球变成 2 颗球，沿机器外框回到上层炮台重新循环；兵种珠通过可见管道以匀速送入下层炮膛。
- 下层四个兵种池使用 `4:3:2:1` 宽度比例，召唤消耗为 `3 / 6 / 10 / 16` 颗弹珠。
- 强化累计 30 次后释放阵营终极技能并清空充能；要塞生命降至 `533 / 1600` 时，每局额外释放一次威力显著更高、拥有独立灾变演出的危急技。
- 单位会自动行军、索敌、攻击和摧毁敌方要塞。界面只保留暂停、`×1/×2/×4` 倍速、静音和重开等观战控制。
- 开局可打开“锦标赛”开关：四阵营随机分成两场半决赛，胜者自动进入总决赛；每场结束自动推进，最终记录三场战报与总冠军，并在短暂停留后开始下一届。届数、阵营胜负、冠军数和最近战报保存到本机浏览器并持续累积。

弹珠、流水线障碍和反弹由 [Matter.js](https://brm.io/matter-js/) 计算。弹珠从真实炮口位置生成，角度在 `20°-80°` 间扫动，初速度随机；结果由最终落点决定，画面和计数使用同一份几何数据。

## 阵营、兵种与技能

| 阵营 | 定位 | 四级兵种 | 终极技能 | 要塞危急技 |
| --- | --- | --- | --- | --- |
| 赤焰军 | 正面突破、持续升温 | 突击兵、长枪兵、狂战士、炎刃统领 | 天火审判 | 炼狱天坠 |
| 霜蓝卫 | 高生命、减速控制 | 盾卫、冰弓手、重甲守军、寒霜巨像 | 绝对零度 | 永冻纪元 |
| 翠核同盟 | 治疗、反伤、持久消耗 | 藤刃斥候、孢子投手、荆棘卫士、古树泰坦 | 森罗生长 | 万木归宗 |
| 鎏金王庭 | 连锁雷击、快速压制 | 电弧侍从、逐光弩手、雷铠先锋、曜日执政官 | 日冕超载 | 恒星耀斑 |

正式战场与兵种图鉴共用同一套侧视分层 SVG 骨架，头盔、胸甲、肩甲、四肢、脚掌和武器均为独立实体部件，并响应行走、攻击、受击和统领施法状态。四级单位使用明显增大的体型、护甲、武器、光环和粒子效果。四名统领还拥有独立技能：炎军号令、冰脉壁垒、森灵复苏和日轮裁决，技能会真实改变单位生命、护盾、速度、控制或要塞生命。

远程单位的冰箭、孢子和棱光弹会从手上武器的挂点出射，在目标胸口高度命中，并使用阵营专属弹体与尾迹。

## 锦标赛与开发测试工具

锦标赛战报面板显示当前两场半决赛、总决赛签位和实时晋级状态。平衡性统计按阵营记录总胜负、左右侧胜负、六组对手交锋；每场明细保留左右阵营、胜方、结束血量、出兵数和耗时，历届记录也明确标记双方站位。开始界面提供二次确认的“清空统计”按钮；关闭锦标赛开关后仍按原来的单场抽签流程运行。

测试工具默认完全隐藏。双击左上角游戏标题，或按 `Alt+Shift+D`，可显示顶栏 `T` 按钮并打开测试台。测试台可自定义左右阵营并跳过抽签直接开局，也支持双方 I-IV 级定向召唤、手动释放统领技能、强化终极技和要塞危急技，以及一键清空全场士兵。测试召唤不消耗弹珠池，并从各自基地正常出发。

## 本地运行

这是一个无后端静态网页，可以直接打开 `index.html`，也可以在项目目录启动本地服务器：

```powershell
python -m http.server 8765 --bind 127.0.0.1
```

然后访问 `http://127.0.0.1:8765/`。

BGM 文件为 `Waterflame - Glorious Morning.mp3`，游戏默认静音，观众可以通过顶栏音乐按钮主动开启。

## 项目结构

```text
.
|-- index.html                 # 页面结构与开始抽签界面
|-- styles.css                # UI、单位动作、弹道与技能特效
|-- game.js                   # Matter.js 弹珠、出兵、战斗与技能逻辑
|-- DESIGN.md                 # 当前玩法、数值和视觉设计文档
|-- unit-art-preview.html     # 统领与兵种独立视觉预览
|-- vendor/                   # Matter.js 及其许可证
|-- remotion/                 # 大炮弹珠视频合成，不参与实时物理
|-- package.json              # Remotion 与 TypeScript 命令
`-- Waterflame - Glorious Morning.mp3
```

## Remotion 辅助素材

`remotion/` 只用于生成视频或静帧素材，不负责网页中的实时刚体计算。

```powershell

npm run still:cannon
npm run typecheck
```
